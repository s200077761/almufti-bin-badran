================================================================================
                        كيفية الاستخدام
                      HOW TO USE
================================================================================

1. التثبيت:
   pip install almufti-bin-badran

2. الاستخدام من سطر الأوامر:
   almufti chat          # محادثة تفاعلية
   almufti search "query"  # بحث
   almufti math "2x+5=15"  # حل مسألة
   almufti report        # تقرير الأداء

3. الاستخدام البرمجي:
   from almufti.core.chat_engine import ChatEngine
   chat = ChatEngine()
   response = chat.generate_response("السلام عليكم")

4. واجهة الويب:
   python app.py
   ثم افتح http://localhost:7860
